# About Project

"Connected" - это приложение, в котором реализован функционал для общения в текстовом формате, используя Вебсокеты

## Основные фичи включают:

- **Регистрация/Авторизация**
- **Поиск чатов и пользователей**
- **Создание чатов при клике найденных пользователей**
- **Отправка/получение сообщений в чатах**

## Папки/Файлы на которые нужно обратить внимание:

- **docker/** - содержит файлы для настройки Docker-контейнеров
- **resources/** - содержит файлы frontend части
- **routes/** - содержит файлы для определения маршрутов приложения (API)
- **database/** - содержит файлы для настройки базы данных и миграций
- **database/seeders/** - содержит файлы для заполнения базы данных тестовыми данными
- **app/Http/Controllers/** - содержит контроллеры для обработки запросов и ответов (обычно добавляют сервисы и репозитории для разделения логики, но проект небольшой, сделано для упрощения навигации)
- **app/Models/** - содержит модели для работы с базой данных
- **app/Events/** - содержит события для работы с базой данных

## Для запуска проекта:

1. Запустите команду `docker compose build && docker compose up -d`.
2. Подождите 20-30 секунд (может быть ошибка от Nginx временная)
3. Откройте два браузера и перейдите по адресу `http://localhost:8000` для доступа к приложению.

Вся необходимая конфигурация настроена и автоматически подгружена на уровне старта контейнеров.

Некоторые пользователя уже были созданы в **app/database/seeders/**, можно посмотреть их креды и ввести на странице Login
